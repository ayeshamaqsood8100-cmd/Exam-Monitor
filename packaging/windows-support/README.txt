MARKAZ SUPPORT RESET

Use this only if an invigilator or IT support asks you to reset Markaz.

What it does:
- closes running Markaz processes
- removes saved Markaz session files
- removes the local blocked marker
- removes Markaz auto-start entries

How to use it:
1. Close Markaz if it is visible.
2. Double-click "Reset Markaz.bat".
3. Wait for the reset to finish.
4. Open Markaz.exe again.

Staff reference:
The PowerShell reset script in this folder performs the same cleanup as the
approved manual reset command for Markaz local state.
